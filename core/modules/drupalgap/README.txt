DrupalGap is an open source application development kit for Drupal powered by
Angular JS.

# Installation and Setup

0. Download the DrupalGap module to the "modules" or "modules/contrib" folder.

1. Enable the DrupalGap module under the "Extend" menu

2. Enable the development version of the REST UI module:

  https://www.drupal.org/project/restui

3. Go to admin/config/services/rest and enable your desired resources.
